## Các loại hình cuộc họp được hỗ trợ

NoteX AI hoạt động trên mọi định dạng. Chuyển đổi giữa các nền tảng một cách dễ dàng.

<Tabs>
  <Tab title="Trực tuyến" icon="video">
    Zoom, Google Meet, Microsoft Teams, Webex. Tự động tham gia qua tiện ích trình duyệt hoặc ứng dụng máy tính.
  </Tab>

  <Tab title="Trực tiếp" icon="users">
    Sử dụng micro laptop hoặc kết nối thiết bị chuyên dụng. Lý tưởng cho phòng họp hoặc hội thảo.
  </Tab>

  <Tab title="Kết hợp" icon="monitor-speaker">
    Kết hợp nguồn âm thanh trực tuyến và tại chỗ. Đảm bảo ghi lại tất cả mọi người, bất kể ở đâu.
  </Tab>
</Tabs>

## Bắt đầu nhanh

Thiết lập và sử dụng chỉ trong vài phút.

<Steps>
  <Step title="Đăng ký tài khoản" icon="user-plus" title-type="p">
    Tạo tài khoản miễn phí tại <a href="https://notexapp.com/home" target="_blank" rel="noopener noreferrer">https\://notexapp.com/home</a> - Không cần thẻ tín dụng.
  </Step>

  <Step title="Cài đặt tiện ích" icon="download" title-type="p">
    Thêm tiện ích trình duyệt NoteX AI từ Chrome Web Store hoặc tải ứng dụng máy tính.
  </Step>

  <Step title="Tham gia cuộc họp" icon="video" title-type="p">
    Bắt đầu hoặc tham gia bất kỳ cuộc họp nào được hỗ trợ. NoteX AI tự động kích hoạt và bắt đầu ghi âm.
  </Step>

  <Step title="Xem kết quả" icon="file-text" title-type="p">
    Truy cập bản ghi âm, bản dịch và bản tóm tắt trong bảng điều khiển tại [https://dashboard.example.com](https://dashboard.example.com).
  </Step>
</Steps>

Để tích hợp API, sử dụng ví dụ này để lấy bản tóm tắt cuộc họp:

<CodeGroup show-lines="true" tabs={[]}>
  ```javascript
    const response = await fetch('https://api.example.com/v1/meetings/{MEETING_ID}/summary', {
      headers: {
        'Authorization': `Bearer ${YOUR_API_KEY}`
      }
    });
    const summary = await response.json();
    console.log(summary.actions);
  ```

  ```python
    import requests

    response = requests.get(
      'https://api.example.com/v1/meetings/{MEETING_ID}/summary',
      headers={'Authorization': f'Bearer {YOUR_API_KEY}'}
    )
    summary = response.json()
    print(summary['actions'])
  ```
</CodeGroup>
